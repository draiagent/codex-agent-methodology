# Advanced AI Cognitive Framework Skill

這是一個面向 Codex 的進階 AI 認知與風險診斷技能，用來把「AI 為什麼會錯」升級成可執行的分析、驗證、治理與系統設計流程。

## 適用場景

- AI 課程與能力地圖設計
- Prompt 與 Context 診斷
- RAG、Agent、MCP、Memory 架構評估
- 企業 AI 導入成熟度分析
- AI 輸出查核與風險治理
- 將概念轉成 SOP、規格或 GitHub 文件

## 啟動方式

可對 Codex 說：

```text
使用 advanced-ai-cognitive-framework，分析這個 AI 工作流的風險、缺失與改善方案。
```

或：

```text
使用進階 AI 認知框架，先盤點已知、未知、假設與待驗證，再給我最小修正與系統性修正。
```

## 核心輸出

1. 任務與風險分級
2. AI 認知層級診斷
3. 已知／未知／假設／待驗證
4. 根因分析
5. 最小修正
6. 系統性修正
7. 驗證與治理
8. Next Action

## 設計理念

```text
模型能力 × 上下文品質 × 工具能力 × 驗證機制 × 人類治理
= 可交付的 AI 系統
```

單純增加模型能力，不等於能產出可信結果。真正的品質來自完整的證據鏈、工具鏈與責任鏈。
